# vs code extensions recommended 

For **MERN + Next.js** development, you don't need dozens of extensions. Around **10–15 high-quality extensions** are enough for a professional workflow.

## Must Have ⭐⭐⭐⭐⭐

| Extension                     | Why                                                                  |
| ----------------------------- | -------------------------------------------------------------------- |
| **ESLint** (Microsoft)        | Shows lint errors and auto-fixes code.                               |
| **Prettier - Code formatter** | Formats code consistently.                                           |
| **Error Lens**                | Displays errors/warnings inline instead of only in the Problems tab. |
| **GitLens**                   | Powerful Git history, blame, and code insights.                      |
| **Tailwind CSS IntelliSense** | Autocomplete, hover previews, and class validation for Tailwind.     |
| **Auto Rename Tag**           | Renames paired HTML/JSX tags automatically.                          |
| **Path Intellisense**         | Autocompletes file import paths.                                     |
| **npm Intellisense**          | Autocompletes npm package imports.                                   |

---

## Very Useful ⭐⭐⭐⭐☆

| Extension               | Why                                                                                  |
| ----------------------- | ------------------------------------------------------------------------------------ |
| **Thunder Client**      | Lightweight API testing inside VS Code (good alternative to Postman for many tasks). |
| **DotENV**              | Syntax highlighting for `.env` files.                                                |
| **MongoDB for VS Code** | Browse MongoDB databases and collections from VS Code.                               |
| **Docker**              | Useful if you're using Docker for development or deployment.                         |
| **Code Spell Checker**  | Catches typos in variable names, comments, and documentation.                        |

---

## React / Next.js Specific ⭐⭐⭐⭐☆

| Extension                                  | Why                                                 |
| ------------------------------------------ | --------------------------------------------------- |
| **ES7+ React/Redux/React-Native Snippets** | Fast snippets like `rafce`, `useState`, etc.        |
| **Auto Import - ES6, TS, JSX, TSX**        | Suggests and inserts missing imports automatically. |

---

## Optional but Nice ⭐⭐⭐☆☆

| Extension                    | Why                                                              |
| ---------------------------- | ---------------------------------------------------------------- |
| **Material Icon Theme**      | Better file and folder icons.                                    |
| **Pretty TypeScript Errors** | Makes TypeScript errors easier to read.                          |
| **Todo Tree**                | Finds all `TODO`, `FIXME`, and similar comments in your project. |

---

2. Prettier
3. Error Lens
4. GitLens
5. Tailwind CSS IntelliSense
6. Path Intellisense
7. Auto Rename Tag

11. DotENV
12. ES7+ React Snippets
13. Material Icon Theme

This combination covers formatting, linting, Git, API testing, database access, Tailwind, React/Next.js development, and general productivity without adding unnecessary extensions.
